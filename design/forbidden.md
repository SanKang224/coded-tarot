# 금지 패턴 목록 — CODED TAROT (부록 M)

## 시각 디자인 금지
- 파스텔 컬러 (핑크, 라벤더, 민트 등)
- CSS 그라데이션 (linear-gradient, radial-gradient) — 단색 원칙
- 배경색 변경 (#000000 절대 유지)
- 타로 카드 이미지 렌더링 (텍스트 전용)
- 둥근 버튼 UI (터미널에는 버튼 없음)
- 아이콘 라이브러리 사용

## AI 문체 금지
- ~해요 / ~습니다 / ~요 어미
- 위로, 공감, 응원 표현
- 타로 결과 선예측 (질문 고도화 단계에서)
- 두 개 이상의 섹터 동시 처리
- 모호한 격려 ("잘 될 거예요" 류)

## 코드 금지
- 불필요한 추상화 (1회용 코드에 클래스 계층 금지)
- 요청되지 않은 기능 추가
- 외부 이미지 CDN 의존
- `any` 타입 남용 (TypeScript)

## UX 금지
- 팝업 / 모달 (터미널 감성 파괴)
- 마우스 호버 색상 변경 이외의 애니메이션 과잉
- 자동 스크롤 이외의 페이지 이동
