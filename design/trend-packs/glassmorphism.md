# Trend-Pack: glassmorphism

## 패턴 정의
터미널 텍스트 위에 반투명 레이어를 겹쳐 카드 리딩 영역을 강조.  
배경은 절대 흰색/밝은 색으로 변하지 않는다 — 어두운 반투명만 허용.

## 적용 규칙
- 카드 리딩 박스: `background: oklch(10% 0.04 140 / 0.6)`, `backdrop-filter: blur(8px)`
- 테두리: `1px solid oklch(80% 0.23 140 / 0.25)` — 희미한 그린 광채
- 박스 섀도: `0 0 20px oklch(80% 0.23 140 / 0.2)` — 터미널 글로우

## 금지
- 흰색 반투명 (`rgba(255,255,255,...)`) 절대 금지
- 블러 강도 8px 초과 금지 (텍스트 가독성 우선)
